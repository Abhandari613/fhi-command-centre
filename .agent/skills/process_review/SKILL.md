# Process Review Skill

This skill allows the agent to adopt four distinct business personas to critique, stress-test, and refine any given process, workflow, or feature implementation.

## Personas

### 1. Sales / CRM ("The Deal Closer")

**Focus:** Speed, Momentum, Ease of Closure.
**Key Questions:**

- "Does this add friction to the deal?"
- "What happens if the client goes silent?"
- "Is the value proposition clear at every step?"
- "How fast can I get the contract signed?"

### 2. Operations / Field Service ("The Site Captain")

**Focus:** Execution, Reality, Logistics.
**Key Questions:**

- "What happens when things go wrong on-site?"
- "Is this practical for a guy in a truck using a phone?"
- "Do we have the materials/info we need _before_ we drive there?"
- "Are we preventing wasted trips?"

### 3. Finance ("The Bean Counter")

**Focus:** Cash Flow, Risk, Verification.
**Key Questions:**

- "Is the money actually in the bank?"
- "Are we exposed to non-payment risk?"
- "Can we track this in the P&L?"
- "Is the deposit verified before work starts?"

### 4. Customer Experience ("Client Success")

**Focus:** Clarity, Trust, Simplicity.
**Key Questions:**

- "Is the client overwhelmed with emails?"
- "Do they know what happens next?"
- "Is the tone friendly and professional?"
- "Are we asking them to do too much work?"

## Usage

When invoked, the agent should:

1.  **Analyze** the proposed workflow/feature.
2.  **Simulate** each persona's critique.
3.  **Synthesize** the findings into a "Stress Test Report".
4.  **Recommend** actionable improvements to mitigate identified risks.
